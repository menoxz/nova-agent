# `read_excel` / XLSX reader

`read_excel` inspects Excel `.xlsx` workbooks without requiring Microsoft Excel or LibreOffice.

It is intended for analysis-oriented reads: workbook structure, sheet dimensions, ranges, formulas, comments/notes, hyperlinks, text search, and detectable tables/images.

## Supported file type

- Supported: `.xlsx`
- Not supported: legacy `.xls` — convert to `.xlsx` first
- Max file size: 100 MB

## Modes

```ts
mode:
  | "metadata"
  | "sheets"
  | "range"
  | "formulas"
  | "comments"
  | "hyperlinks"
  | "tables"
  | "images"
  | "search"
  | "all"
```

Default mode: `sheets`.

## Examples

### Workbook metadata

```json
{ "path": "report.xlsx", "mode": "metadata" }
```

Returns file size, sheet count, title, subject, creator, modified dates, company, date1904/calc metadata, and defined names when present.

### Sheet map and dimensions

```json
{ "path": "report.xlsx", "mode": "sheets" }
```

Returns every sheet with visibility, actual/declared row and column counts, used range, merge count, table count, and image count.

### Read a range

```json
{
  "path": "report.xlsx",
  "mode": "range",
  "sheet": "Data",
  "range": "A1:D20",
  "maxRows": 100,
  "maxCols": 30
}
```

For each cell, returns available value plus formula, cached formula result, note/comment, and hyperlink.

### Search text

```json
{ "path": "report.xlsx", "mode": "search", "query": "Nova" }
```

Searches across visible cell values, formulas, formula results, notes/comments, and hyperlinks.

### Inspect formulas only

```json
{ "path": "report.xlsx", "mode": "formulas" }
```

Returns formula cells grouped by worksheet, including cached results when available.

## Formula behavior

The tool reads formulas and cached results stored inside the workbook. It does **not** recalculate formulas. If a workbook was generated by a library and never opened/recalculated by Excel/LibreOffice, cached formula results may be missing or stale.

## Tables and images

`read_excel` reports ExcelJS-detectable worksheet tables and embedded worksheet images:

- tables: name/display name, reference, totals row flag, columns
- images: workbook image id, extension, approximate worksheet range, byte/base64 size when available

## Limits / future work

- No `.xls` support.
- No formula recalculation.
- Charts, pivots, slicers, macros/VBA, PowerQuery, external connections and rich formatting extraction are not yet exposed.
- Image coordinates are converted best-effort from ExcelJS anchors.
