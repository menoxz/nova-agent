# MCP Prompts

Registered prompts:

- `nova_repository_orientation` — read-only repository orientation workflow.
- `nova_readonly_review` — read-only code/docs review workflow.
- `nova_tool_safety_review` — review the MCP tool safety posture.
- `nova_eval_scenario_design` — design safe deterministic eval scenarios.
- `nova_trace_summary_diagnosis` — diagnose behavior from sanitized trace summaries only.
- `nova_mcp_client_setup` — explain client and Inspector setup.

Prompts are guidance templates only; they do not grant write, shell, or raw artifact access.
