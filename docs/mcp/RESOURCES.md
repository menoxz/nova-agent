# MCP Resources

Nova MCP exposes curated resources only. It does not expose a raw filesystem mirror.

## Resource URIs

- `nova://docs/status`
- `nova://docs/mcp/readme`
- `nova://docs/mcp/tools`
- `nova://docs/mcp/security`
- `nova://docs/mcp/resources`
- `nova://docs/mcp/prompts`
- `nova://docs/mcp/client-setup`
- `nova://tools/catalog`
- `nova://eval/scenarios`
- `nova://eval/schema`

Resources are documentation or generated metadata summaries. Sensitive local artifacts remain denied through file tools and are not mirrored as resources.
